## Начало работы

Запуск сервера разработки:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Открыть [http://localhost:3000](http://localhost:3000) в браузере.

## Проверка типов
```bash
# Проверка
npm run lint

# Исправление
npm run lint:fix
```

## Проверка стиля кода
```bash
# Проверка
npm run lint

# Исправление
npm run lint:fix
```

## Проверка типов в режиме реального времени
```bash
npm run ts-lint
```
